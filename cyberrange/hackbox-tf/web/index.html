Rutgers Cyber Range

Welcome to the Rutgers Cyber Range, a platform for students to learn and practice cybersecurity skills in a safe and controlled environment.

Features

  • AWS-based cloud infrastructure
  • Autoscaling EC2 instances
  • Linux-based operating system
  • Docker container isolation for users
  • Common Vulnerability Exploits (CVEs) for penetration testing education
  • SSH access and password reset functionality

Activities

  • Find running services and ports
  • Investigate Nginx web server
  • Install ansible and create YAML playbook
  • Use Nmap to find open ports
  • Attempt to break out of Docker container (feasibility not yet determined)
